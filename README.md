# Extension_Media_Bias
A simple Google Chrome extension designed to detect and evaluate the political bias of a specific news article whenever the user navigates onto a valid page. Simply click on the extension icon whenever traveling to a page in order to check its media bias value.

The media bias values are taken from a cleaned database that lists the average bias values of known news site. The size of this dataset is robust and capable of detecting a multitude of websites, including those that are less mainstream. As such, this extension can be used for a variety of situations but should not be used as the sole determiner for a news site's reliability. It is important to remember that this tool should serve as an aid to whenever possible.

There are plans to expand upon this project and add more self sustainable features such as keyword matching and comparisons across news articles, as well as more dynamically adjusted databases in order to accomodate for the changing landscape of the internet and news corporations. In addition, there will be changes to the overall structure of the extension to detect not just political bias, but also informational bias and other metrics that should prove to be a stronger determinant for how reliable a particular news source is.
